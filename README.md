# amxx-teamswitch
Automate team switch in a competitive environment (Per @Carl's view).

# Console Variables
mp_game_max_rounds  (Default: 15)  | Amount of rounds total that can be played. If no team reaches mp_game_win_limit.
mp_game_half_time   (Default: 7)   | Amount of rounds to switch teams. Half time.
mp_game_win_limit   (Default: 8)   | Amount of rounds needed to win to win the game.
